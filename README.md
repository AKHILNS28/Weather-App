# 🌦️ Weather App

A simple weather application that allows users to search for a city and view real-time weather information like temperature, humidity, and weather conditions.

---

## Features

- Search weather by city name  
- Displays temperature (°C)  
- Shows humidity  
- Weather condition description  
- Emoji representation of weather  
- Error handling for invalid input  

---

## Technologies Used

- HTML  
- CSS  
- JavaScript  
- OpenWeather API  

---

## ▶️ How to Run

1. Download or clone the repository  
2. Open `index.html` in your browser  
3. Enter a city name and get weather details  

---
